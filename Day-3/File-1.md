# Linux Commands:-

* **sudo su** -> $ to # 

* **exit** -> # to $

* **ls** -> list directory files

* **pwd** -> print working directory

* **~** -> represents home directory of current user

* **cd** -> change directory

* **/** -> represents root

* **clear** -> clear screen

* **mkdir** -> create directory

* **vi** -> tool/program(works like notepad)

* **i** -> insert 

* **:w** -> write in the file

* **:q** -> exit from the current file
